# rand-id

> a simple uid generator

## Install

```
$ npm install --save rand-id
```

## Usage

```js
const randId = require('rand-id');

randId(20);
//=> 'JzVfNN3GWuv4LxJegFQI'
```

## API

### randId(length)

#### length

Type: `number`

Number of random characters.
